is_square_matrix Function (ROM Call 0x3B8)
AMS 1.01 or higher estack.h

short is_square_matrix (CESI ptr);

Checks whether an expression is a square matrix.

is_square_matrix returns TRUE if the expression pointed to by ptr is a square matrix, otherwise it returns FALSE.

Uses: is_matrix, remaining_element_count
Used by: push_acosh, push_asin, push_asinh, push_atan, push_atanh, push_cosh, push_eigvc, push_eigvl, push_exp, push_ln, push_log10, push_sin2, push_sinh, push_tan, push_tanh, push_exponentiate, push_list_plus
