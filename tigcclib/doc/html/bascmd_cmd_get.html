cmd_get Function (ROM Call 0x350)
AMS 1.01 or higher bascmd.h

void cmd_get (SYM_STR SymName);

Executes TI-Basic 'Get' command.

cmd_get receives a CBL (Calculator-Based Laboratory) value from the link port and stores it in a variable whose name is pointed to by SymName (a VAT variable name, see SYMSTR).

Uses: HeapAllocThrow, HeapFree, HeapLock, HeapUnlock, push_assignment, push_indir_name, ER_catch, ER_success, ER_throwVar, HToESI, push_expression, push_Float, top_estack, LIO_Get, memcpy, round14, primary_tag_list, store_to_subscripted_element, DerefSym, VarRecall, VarStore, _du16u16, ROM Call 0x4D4

See also: cmd_send, cmd_getcalc
